Chupa cuatro
El siguiente jugador debe robar cuatro cartas del mazo. El jugador que debe robar las cartas puede responder con otra carta Chupa cuatro o con una carta Chupa dos para aumentar el número de cartas totales que deben ser robadas y pasar el turno al siguiente jugador.
{{ 'Interacciones especiales' | translate }}
  • La carta Parry puede actuar como un cambio de sentido y devolver la cadena de cartas totales que deben ser robadas.
  • La carta Thanos - Gema azul disminuir a la mitad el número de cartas totales que deben ser robadas.
  • Esta carta no puede ser denegada por una carta Turno prohibido ni por una carta Cambiar sentido.
